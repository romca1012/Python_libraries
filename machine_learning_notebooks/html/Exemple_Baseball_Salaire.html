Prédiction du salaire des nouveaux membres d'une équipe de baseball américain à l'aide d'un arbre de décision¶

Une équipe de baseball décide de créer un modèle de machine learning afin d'estimer les salaires des nouveaux joueurs recrutés en fonction de leurs performances et de leur expérience. Utilisez un modèle d'arbre de régression pour résoudre ce problème à l'aide de Pycaret.

  • D'après l'arbre de décision obtenu, quelle est la variable la plus déterminante pour prédire le salaire des joueurs

L'équipe veut engager quatre nouveaux joueurs ayant les performances et années d'expérience suivantes : Années_experience:[5, 7, 10, 15], Performance_Hits:[230, 100, 120, 200].

  • Quels devraient être les salaires des nouveaux joueurs ? Pour ce faire, utilisez la fonction « Predict » et la règle conditionnelle reprogrammée.

import numpy as np

from pycaret.regression import *

data_baseball =pd.read_excel('data/baseball_data.xlsx')  # Lecture de données
data_baseball_selected = data_baseball[['Salary','Hits','Years']]  # Sélection des varaibles d'intérêt
data_baseball_selected['Salary'].fillna(data_baseball_selected['Salary'].mean(), inplace=True)  # Imputation de la variable cible
data_baseball_selected['Salary'] = np.log(data_baseball_selected['Salary'])  # Scaling de la variable salaire (millions) 

# Initialisation de l'environnement pycaret
exp = setup(
    data_baseball_selected, 
    target='Salary', 
    session_id=123,

# Création de la grille d'optimisation pour le modèle d'arbre de décision
dt_grid_baseball_reg = {

    #'criterion': ['squared_error']  # Définition du critère d'optimisation des variables

}

dt_model_baseball_reg = create_model('dt', random_state=123)  # Création du modèle d'arbre de régression

tuned_dt_baseball_reg, tuner_dtbaseball__reg = tune_model(

    fold = 5,
    return_tuner=True, 
    return_train_score=True
)  # n_iter doit être les même que les nombre d'arbres max_depth pour afficher tous les résultats de validation croisée
cv_results_dt_baseball_reg =pull()  # Extraction des résultats de la validation croisée pour dt

final_model_baseball_reg = finalize_model(tuned_dt_baseball_reg)  # Choix du modèle optimal

# Réalisation des graphiques d'arbre de décison et de l'importance des variables
plot_model(final_model_baseball_reg.named_steps['actual_estimator'], plot='tree')

# Définition de la fonction de règle de décision (pour l'arbre de régréssion
def predict_value(Years, Hits):
    if Years <= 4.50:
        if Hits <= 36.50:
            if Hits <= 1.50:
                return 7.66
            else:
                return 6.23
        else:
            if Years <= 3.50:
                return 5.01
            else:
                return 5.72
    else:
        if Hits <= 117.50:
            if Years <= 6.50:
                return 5.78
            else:
                return 6.17
        else:
            if Hits <= 224.50:
                return 6.69
            else:
                return 7.59

# Utilisatition de la fonction predict de pycaret
