Prédiction des salaires des professionnels dans différents secteurs à l'aide du modèle Random Forest (SciKitLearn)¶

Un bureau d'études a été engagé pour créer un modèle permettant de prédire le salaire des salariés dans divers secteurs. Pour ce faire, il a reçu un jeu de données contenant des informations sur le genre, le salaire, le niveau de scolarité, les années d'expérience et l'âge. Par ailleurs, l'étude doit viser à déterminer quels sont les facteurs les plus déterminants dans les salaires. Les membres de l'équipe d'étude ont choisi d'utiliser un modèle Random Forest pour résoudre ce problème et doivent fournir des réponses rapides aux questions suivantes:

  • Quel ensemble d'hyperparamètres permet d'obtenir la meilleure métrique?
  • Quels sont les variables les plus déterminantes pour la définition du salaire?
  • Existe-t-il une différence de salaire en fonction du genre?
  • Présenter un graphique montrant la dispersion des valeurs prédites pour le modèle et les valeurs réelles.

# Import des bibliothèques
import numpy as np
import pandas as pd
from sklearn.model_selection import train_test_split, KFold, GridSearchCV
from sklearn.preprocessing import OneHotEncoder, LabelEncoder
#from sklearn.tree  import DecisionTreeRegressor, plot_tree, export_text
from sklearn.metrics import r2_score
from sklearn.ensemble import RandomForestRegressor
#from sklearn.svm import SVR
import matplotlib.pyplot as plt
import seaborn as sns

data_salary =pd.read_csv('data/Salary_Data.csv')  # Lecture des données 
data_salary = data_salary.dropna(axis=0)  # Suppression des valeurs manquantes
cate_var_salary = data_salary.select_dtypes(['object','category'])  # Sélection de variables catégorielles selon le dtype
num_var_salary = data_salary.select_dtypes(['int','float'])   # Sélection de variables numériques selon le dtype
encod_cat = OneHotEncoder(sparse_output=False)  # Définition d'objet OnehotEncoder
var_cat_encoded_salary = encod_cat.fit_transform(cate_var_salary)   # Transformatin des variables catégorielles

df_salary_encode_cat = pd.DataFrame(
    var_cat_encoded_salary, 
    columns=encod_cat.get_feature_names_out(), 
    index=data_salary.index
)  # Conversion du data frame des variables catégoreilles encodées. Lorsqu'on fait la conversion en data frame il faut uiliser 
# les indices originaux du data frame afin d'éviter des problèmes dans la concatenation à cause des changements des indices dans l'Onehot encode

df_encode_num_cat = pd.concat(
    [num_var_salary, 
    df_salary_encode_cat], 
    axis=1
)

# Division des varaibles déscriptives et la variable cible
X_encod_num_cat = df_encode_num_cat.drop('Salary', axis =1)
y_encod_num_cat =  df_encode_num_cat['Salary']

X_encod_num_cat_train, X_encod_num_cat_test, y_encod_num_cat_train, y_encod_num_cat_test = train_test_split(X_encod_num_cat, y_encod_num_cat,random_state=42, test_size=0.3)

model_rf_salary =RandomForestRegressor(random_state=42)

param_search_opt_salary = {
    'n_estimators':[100, 200, 500, 700],
    'max_depth': [None, 10, 20, 30],
    #'min_samples_split': [2, 5, 10],
    #'min_samples_leaf': [1, 2, 4],
    'bootstrap': [True, False]
}
grid_search_rf_salary = GridSearchCV(
    estimator=model_rf_salary, 
    param_grid=param_search_opt_salary
)

grid_search_rf_salary.fit(X_encod_num_cat_train, y_encod_num_cat_train)

print(f"Le coefficient de détermination (r2) optimisé est : {r2_score(y_encod_num_cat_test, grid_search_rf_salary.best_estimator_.predict(X_encod_num_cat_test))}")

print(f"Les hyper-paramètres optimisés pour le modèle sont:{grid_search_rf_salary.best_params_}")

# Réalisation du graphique données prédites vs données réelles
plt.scatter(
    y_encod_num_cat_test,
    grid_search_rf_salary.best_estimator_.predict(X_encod_num_cat_test))

# Définissant les limites du graphiques pour la droite identité
x_min, x_max = plt.xlim()
y_min, y_max = plt.ylim()

# Limites des axes
axis_min = min(x_min, y_min)
axis_max = max(x_max, y_max)

# Création du graphique indentité
identity_line = np.linspace(axis_min, axis_max)
plt.plot(identity_line, identity_line, 'k-', label='')
plt.xlabel('Valeurs réelles')
plt.ylabel('Performance prédite')
plt.show()


# Graphique d'importance des variables Random Forest

     'importance':grid_search_rf_salary.best_estimator_.feature_importances_
    }
).sort_values('importance', ascending=False)  # Construction du data frame d'importance 

sns.barplot(x='importance', y='features', data=importance.iloc[range(0,20),:])  # Barplot d'importance avec seaborn
