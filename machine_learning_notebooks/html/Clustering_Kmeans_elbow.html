Application de la méthode du coude (elbow) pour déterminer le nombre optimal de clusters à partir de données synthétiques¶

L'objectif de cet exercice est de démontrer l'utilisation de la méthode du coude (elbow) pour déterminer le nombre optimal de clusters. Cet exemple a été créé à partir de données synthétiques afin d'illustrer l'efficacité de cette méthodologie. La fonction make_blobs() génère des points de données répartis autour de plusieurs centres dans l'espace multidimensionnel. Chaque centre représente l'emplacement d'un cluster de points de données.

import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
from sklearn.datasets import make_blobs

# Générer un jeu de données de données synthétiques avec make_blobs.
# X, _ signifie que je n'aurais besoin des étiquettes des clusters, uniquement des variables de l'array X 
X, _ = make_blobs(n_samples=300, centers=4, random_state=42)

# Calculer l'inertie pour différentes valeurs de K
inertia = []
K_range = range(1, 11)  # Tester pour 1 à 10 clusters
for k in K_range:
    kmeans = KMeans(n_clusters=k, random_state=42)
    kmeans.fit(X)
    inertia.append(kmeans.inertia_)

# Traçage de l'inertie en fonction du nombre de clusters
plt.figure(figsize=(8, 6))
plt.plot(K_range, inertia, marker='o')
plt.title('Méthode du Coude pour choisir le nombre de clusters optimal')
plt.xlabel('Nombre de clusters (K)')
plt.ylabel('Inertie')
plt.show()

# Application du KMeans avec le nombre de clusters optimal (K=4)
optimal_k = 4  # Nous avons choisi 4 à partir de la méthode du coude
kmeans = KMeans(n_clusters=optimal_k, random_state=42)
y_kmeans = kmeans.fit_predict(X)

# Tracer les clusters en 2D
plt.figure(figsize=(8, 6))
plt.scatter(X[:, 0], X[:, 1], c=y_kmeans, s=50, cmap='viridis')

# Marquer les centres des clusters
centers = kmeans.cluster_centers_
plt.scatter(centers[:, 0], centers[:, 1], c='red', s=200, marker='X', label='Centres des clusters')

plt.title(f'Visualisation des Clusters avec K={optimal_k}')
plt.xlabel('X1')
plt.ylabel('X2')
plt.legend()
plt.show()
